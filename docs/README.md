# panicbuffet Documentation

## Overview

This is the documentation for the `panicbuffet` project. It provides information on how to set up, run, and contribute to the project.

## Getting Started

### Prerequisites

- Node.js (>= 14.17)
- npm or yarn

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/your-repo/panicbuffet.git
   cd panicbuffet
   ```

2. Install dependencies:
   ```sh
   npm install
   # or
   yarn install
   ```

3. Start the development server:
   ```sh
   npm run dev
   # or
   yarn dev
   ```
