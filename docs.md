### MVP要約

食べ放題レストラン経営ゲームのブラウザベースのMVP（Minimum Viable Product）は、以下の機能を持つものとします：

* ゲーム画面が表示される
* ピザ、パスタ、ドリンクの在庫ゲージが表示される
* 各料理を補充できる
* 補充するとコストが増える
* 客が自動で来店する
* 客が料理を消費する
* 在庫が少ないと満足度が下がる
* 在庫があると売上が入る
* 制限時間が0になると結果画面に切り替わる
* 結果画面に売上、満足度、廃棄ロス、利益、ランクが表示される

### 使用技術

Vite + TypeScriptで実装されます。HTMLおよびCSSはCanvasまたはDOMベースの軽量描画のために使用され、必要な場合にはlocalStorageを保存します。

### ファイル構成

以下のファイル構成が推奨されます：

* package.json
* index.html
* src/main.ts
* src/style.css
* src/game/types.ts
* src/game/state.ts
* src/game/update.ts
* src/game/render.ts
* docs/progress.md

ただし、分割しすぎて作業が遅くなる場合は、「main.ts」中心で実装しても構いません。

### 実行コマンド

`npm run dev`
